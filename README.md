# Finance-Manager
Приложение для учёта личных доходов и расходов с возможностью категоризации, построения графиков, прогнозирования бюджета и многопользовательским доступом с двухфакторной аутентификацией. Данные хранятся в базе, обмен между клиентом и сервером идёт по сети, а для тяжёлых вычислений (прогноз, отчёты) используется асинхронная обработка.
